INFORMACIÓN Y TRANSPARENCIA

Privacidad del seguimiento Braille

Conoce qué información puede registrarse, para qué se utiliza y cómo se protege durante las prácticas Braille.

Resumen: el seguimiento es opcional. La constancia simbólica puede generarse sin autorizar el registro estadístico. De manera predeterminada se utiliza un identificador anónimo y el nombre solo se registra cuando la persona decide usarlo voluntariamente.

Responsable del tratamiento

El seguimiento de uso es administrado por el responsable del Ecosistema Virtual Accesible, en el marco del módulo Materiales Educativos Accesibles del CREBE "Señor de los Milagros" - Ucayali.

Finalidad del registro

La información autorizada se utiliza exclusivamente para medir la interacción con las prácticas Braille, conocer el nivel de uso de las modalidades disponibles y orientar mejoras educativas, técnicas y de accesibilidad. No se emplea con fines comerciales, publicitarios ni para enviar comunicaciones al participante.

Datos que pueden registrarse

  • Identificador anónimo generado por el navegador.
  • Fecha y hora del registro.
  • Modalidad y nivel de práctica.
  • Cantidad de ejercicios, porcentaje, errores y pistas utilizadas, cuando corresponda.
  • Finalización de la actividad y generación de la constancia simbólica.
  • Tipo general de dispositivo: móvil o escritorio.
  • Nombre, únicamente cuando la persona elige voluntariamente aparecer de forma nominal.

Datos que no se solicitan

No se solicitan correo electrónico, documento de identidad, número telefónico, domicilio, contraseña, información financiera ni datos de contacto.

Consentimiento y participación anónima

La casilla de autorización para el seguimiento estadístico no está marcada de forma predeterminada. Si no se otorga el consentimiento, la constancia simbólica puede generarse normalmente y no se envían datos a la hoja de seguimiento.

Cuando se autoriza el registro, la persona puede continuar de forma anónima. El identificador se conserva localmente en el navegador para estimar el uso recurrente desde ese mismo dispositivo. Puede cambiar si se eliminan los datos del navegador o se utiliza otro equipo.

Almacenamiento y acceso

Los registros autorizados se envían mediante Google Apps Script y se almacenan en una hoja de cálculo de Google Drive. El acceso debe mantenerse restringido al responsable del proyecto y a las personas institucionalmente autorizadas para el seguimiento educativo.

Conservación y eliminación

Los registros se conservarán únicamente durante el tiempo necesario para analizar el uso y mejorar el recurso. La persona que haya utilizado voluntariamente su nombre puede solicitar la revisión o eliminación del registro mediante el canal de contacto institucional, indicando la fecha aproximada y el código de constancia cuando lo tenga disponible.

Participación de menores de edad

La práctica está diseñada como recurso educativo. Cuando se utilice con estudiantes menores de edad dentro de una actividad institucional, el acompañamiento y la autorización aplicable deben ser gestionados por el docente, la familia o el responsable correspondiente. Se recomienda mantener la modalidad anónima cuando no sea necesario identificar al participante.

Constancia simbólica

La constancia tiene carácter educativo y motivacional. No constituye certificado oficial, acreditación académica ni constancia institucional de capacitación.

Actualizaciones del aviso

Este aviso puede actualizarse cuando cambie el funcionamiento del seguimiento o se incorporen nuevas medidas de protección. La versión vigente será la publicada en esta página.
