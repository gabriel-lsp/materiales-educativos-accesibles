Generador de conciencia fonológica

Recurso independiente para trabajar grafema, fonema, sílaba y palabra. Permite generar una ficha imprimible con apoyo visual, especialmente útil para estudiantes que requieren anticipación, estructura y claves visuales.

Ruta de trabajo

Primero se reconoce el grafema, luego se asocia con el fonema, después se forman sílabas directas y finalmente se aplica en una palabra con imagen de apoyo.
