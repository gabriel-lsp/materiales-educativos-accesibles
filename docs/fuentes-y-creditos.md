# Fuentes y créditos

Este documento registra los criterios de reconocimiento de fuentes y créditos del repositorio Materiales Educativos Accesibles.

El proyecto es desarrollado y organizado por Gabriel Berrospi como recurso educativo digital personal orientado a la accesibilidad, la inclusión educativa y la atención a la diversidad.

## Criterios generales

Todo material incorporado al repositorio debe conservar su fuente, autoría o procedencia cuando corresponda.

La organización de recursos, adaptación digital, estructura de navegación, criterios de presentación y disposición pedagógica corresponden al trabajo realizado para este proyecto, salvo que se indique expresamente una fuente externa.

## Recursos externos

Cuando se incorporen imágenes, pictogramas, íconos, fotografías, plantillas, textos, videos, documentos, materiales descargables o recursos de terceros, deberá registrarse su procedencia y respetarse sus condiciones de uso.

Si se utilizan materiales provenientes de entidades públicas, repositorios abiertos, bancos de imágenes, plataformas educativas o autores externos, se debe mantener la referencia a la fuente original cuando corresponda.

## Datos recomendados para registrar una fuente

Se recomienda registrar cada recurso externo con los siguientes datos:

- Nombre del recurso.
- Autor, institución o entidad responsable.
- Enlace o procedencia.
- Tipo de recurso.
- Condición de uso o licencia, si está disponible.
- Fecha de consulta o incorporación.

## Uso de logos o referencias institucionales

En caso de utilizar logos, nombres o referencias de una institución educativa, su uso debe realizarse dentro del marco autorizado y sin presentar dichos elementos como propiedad del proyecto personal.

## Reconocimiento del proyecto

Toda reutilización del repositorio debe conservar el reconocimiento del proyecto y del autor responsable.
