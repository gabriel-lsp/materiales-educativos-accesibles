# Criterios de adaptación

Este documento orienta la elaboración y adaptación de materiales dentro del repositorio Materiales Educativos Accesibles.

Los criterios propuestos buscan favorecer que los materiales sean claros, funcionales, accesibles y pertinentes para distintos estudiantes y contextos educativos.

## Principios generales

Todo material debe responder a una necesidad educativa concreta. Antes de diseñarlo o adaptarlo, se recomienda definir qué se quiere apoyar: comunicación, comprensión, anticipación, rutina, asociación, conciencia fonológica, autonomía, participación u otra habilidad específica.

El material debe ser simple, claro y fácil de usar. Un diseño visual atractivo no garantiza que sea accesible si tiene demasiados estímulos, instrucciones largas o elementos difíciles de comprender.

## Claridad visual

Se recomienda utilizar imágenes, textos o apoyos gráficos que sean reconocibles para el estudiante.

Los elementos visuales deben tener tamaño adecuado, buena separación, contraste suficiente y una organización limpia. Cuando sea posible, se debe evitar la sobrecarga de colores, adornos o información innecesaria.

## Lenguaje sencillo

Las instrucciones deben ser breves, directas y comprensibles. Se recomienda usar palabras comunes y frases cortas.

Cuando el material esté dirigido a estudiantes que requieren mayor apoyo, puede usarse una combinación de imagen, palabra clave y modelado del adulto.

## Adaptación según necesidad de apoyo

Un material puede modificarse según el nivel de apoyo requerido. Algunas adaptaciones posibles son:

- Reducir la cantidad de opciones.
- Aumentar el tamaño de imágenes o letras.
- Usar imágenes más familiares.
- Presentar una actividad por vez.
- Dividir una tarea en pasos pequeños.
- Agregar apoyos visuales de inicio y fin.
- Usar colores para organizar información.
- Incorporar ejemplos antes de la actividad.
- Permitir respuestas señalando, emparejando, verbalizando o manipulando.

## Pertinencia pedagógica

El material debe relacionarse con un objetivo educativo claro. No se recomienda acumular materiales sin definir para qué se utilizarán.

Cada recurso debe poder responder a preguntas básicas: qué habilidad apoya, para quién está pensado, cómo se usa y qué tipo de ayuda puede requerir.

## Revisión del material

Después de usar un material, se recomienda observar si fue comprendido, si mantuvo la participación, si generó frustración o si requirió demasiada ayuda.

Si el material no cumple su finalidad, debe ajustarse antes de volver a utilizarse.

## Registro de adaptaciones

Cuando se realicen adaptaciones importantes, se recomienda registrar el cambio en la bitácora del proyecto, indicando qué se modificó y por qué.
